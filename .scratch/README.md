# Episode Scratch

Per-episode research provenance. Each episode gets its own subdirectory when work begins.

```
.scratch/
├── 01-the-value-shift/
│   └── main.md
├── 03-spiritual-before-temporal/
│   └── main.md
└── ...
```

These files are kept after episodes publish — they're the provenance trail that supports the source verification claims made in the scripts. The teaching agent creates these automatically during Phase 1 (outline) work.
